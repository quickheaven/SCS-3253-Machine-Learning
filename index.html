URL dataset (ISCX-URL2016)

The Web has long become a major platform for online criminal activities. URLs are used as the main vehicle in this domain. To counter this issues security community focused its efforts on developing techniques for mostly blacklisting of malicious URLs.
  • We study mainly five different types of URLs:
  • 1. Benign
  • 2. Spam
  • 3. Phishing
  • 4. Malware
  • 5. Defacement

Challenge

Large number of features

Lack of description on the features

The dataset contains Null, NaN and Inf. values

Model

The dataset was trained using both supervised and unsupervised models.

Unsupervised model: Isolation Forest for usupervised anomaly detection

Supervised models: Random Forest, Decision Tree, SVC (Linear), Logistic Regression, SGD and ADA Boost.
